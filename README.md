# Portfolio de Nahuel Noir

Este es mi portfolio profesional, desarrollado con React, TypeScript, Vite y Tailwind CSS.

Lo diseñé para presentar mi perfil de forma clara, visual y mantenible, con foco en reclutadores y oportunidades laborales. El sitio incluye secciones de Hero, About, Projects y Contact, además de soporte multilenguaje, tema claro/oscuro, navegación interna entre secciones y animaciones suaves.

## Producción

- [noirnahuel.vercel.app](https://noirnahuel.vercel.app)

## Ejecutarlo en local

### Requisitos

- Node.js 20 o superior
- npm

### Instalación

```bash
npm install
```

### Desarrollo

```bash
npm run dev
```

Esto levanta el proyecto en modo desarrollo con Vite.

### Build de producción

```bash
npm run build
```

### Preview local del build

```bash
npm run preview
```

## Scripts disponibles

- `npm run dev`: entorno local de desarrollo
- `npm run build`: compila TypeScript y genera el build de producción
- `npm run preview`: previsualiza el build generado
- `npm run lint`: ejecuta ESLint sobre el proyecto

# Nahuel Noir Portfolio

This is my professional portfolio, built with React, TypeScript, Vite, and Tailwind CSS.

I designed it to present my profile in a clear, visual, and maintainable way, with a strong focus on recruiters and job opportunities. The site includes Hero, About, Projects, and Contact sections, as well as multilingual support, light/dark theme switching, internal section navigation, and subtle motion.

## Production

- [noirnahuel.vercel.app](https://noirnahuel.vercel.app)

## Run locally

### Requirements

- Node.js 20 or higher
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

This starts the project in development mode with Vite.

### Production build

```bash
npm run build
```

### Local preview of the build

```bash
npm run preview
```

## Available scripts

- `npm run dev`: local development environment
- `npm run build`: compiles TypeScript and generates the production build
- `npm run preview`: previews the generated build locally
- `npm run lint`: runs ESLint across the project
